M5

5HP · 二冲程

M5 是超轻便携二冲程机型,单缸 102cc 结构,整机仅 20kg,配手提把搬运轻松。内置 2.5L 油箱即开即用,亦可外接 12L 油箱;标配浅水驱动,是小艇、橡皮艇与帆船辅助动力的理想之选。

规格参数

发动机形式二冲程 单缸
最大输出5 ps（3.7 kW）
排量102 cc
缸径 × 行程55 × 43 mm
最高转速范围4,500 – 5,500 rpm
启动方式手拉启动
操控方式舵柄
换挡前进 - 空挡 - 后退

燃油无铅汽油（87 号及以上）
机油TOHATSU 正品 TCW-3 二冲程机油
润滑方式预混燃油
油箱内置 2.5L + 外接 12L（选配）
整机重量20 kg（最轻配置）
发电量12V / 60W / 5A（选配,电启动机型标配）
标准配置挂挡保护 / 螺旋桨排气 / 浅水驱动 / CD 点火
Features

产品特点

动力 · 可靠

  • 久经验证的二冲程技术,可靠耐用
  • 单缸结构,轻巧紧凑
  • 环形扫气进气,运行平顺、油耗经济
  • CD 点火,启动迅捷

便携 · 操控

  • 手提把设计,搬运轻松
  • 内置油箱,即开即用(可选外接油箱)
  • 舵柄操控,简单直接
  • 多种轴距可选

耐用 · 防腐

  • 海洋铝合金 + 内水道镀锌,强抗腐蚀
  • 铝制螺旋桨
  • 预混燃油润滑
  • 浅水驱动(部分机型)

安全保护

  • 安全拉绳(全系标配)
