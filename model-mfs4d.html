MFS4D

4HP · 四冲程

MFS4D 为小型船艇提供可靠动力,单缸四冲程结构,排放更清洁。顶置内置油箱即开即用,亦可外接油箱;前后大提手与 3 向存放设计,便携实用。

Features

产品特点

设计与便携

  • 全新上下盖罩设计,轻巧紧凑
  • 前后大提手,搬运便利
  • 3 向存放,无需担心漏油
  • 同级首创前置换挡杆
  • 超低排放,EPA / CARB 三星认证

操作与可靠

  • 数字 CD 点火,启动更快
  • 大盘绕绳手拉启动,省力
  • 浅水驱动、多挡仰角
  • 加长舵柄,操控更顺手
  • 螺旋桨排气,运行更安静

耐用 · 防腐

  • 恒温冷却系统
  • 不锈钢水泵壳衬套
  • 海洋铝合金 + 内水道镀锌,强抗腐蚀

安全保护

  • 低油压指示
  • 安全拉绳(全系标配)
  • 挂挡启动保护
  • 超转速限制
