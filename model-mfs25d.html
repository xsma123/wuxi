MFS25D

25HP · EFI 四冲程

MFS25D 为充气艇、RIB、铝合金渔船、FRP 及浮筒船等多种船型设计,配置丰富。全新设计实现同级最轻(约 58kg)与更高可靠性;电子燃油喷射、优化齿轮比(2.17:1)、浅水驱动与 17.5A 大功率充电系统,操控灵活高效。

Features

产品特点

设计与性能

  • 全新简约外观,适配充气艇 / RIB / 铝合金渔船 / FRP / 浮筒船
  • 电子燃油喷射(EFI),油门响应迅捷
  • 同级出色的功率重量比,轻量化设计
  • 横流式进气,运行平顺;超低排放(EPA / CARB 认证)

高性能 · 易操作

  • 电子燃油喷射(EFI)
  • 数字 CD 点火,启动更快
  • 17.5A 大功率充电系统
  • 优化齿轮比(2.17:1),低速扭矩强
  • 可调转向阻尼与配平片,转向更省力
  • 浅水驱动,多挡仰角(部分机型)

便利 · 可靠

  • 驱动轴壳内置淡水冲洗,易于保养
  • 免工具透明燃油滤清器,带浮子水检测
  • 螺旋桨排气,运行更安静
  • 恒温冷却系统
  • 不锈钢水泵壳衬套 + 内水道镀锌,强抗腐蚀
  • 配 25L 分离式油箱

安全保护

  • 低油压报警(声光)
  • 过热报警(声光)
  • 挂挡启动保护
  • 安全拉绳(全系标配)
  • 超转速限制
